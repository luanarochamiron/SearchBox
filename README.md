# Responsive Search Box

Uma barra de busca (Search Box) moderna, minimalista e totalmente responsiva desenvolvida com HTML5 e CSS3. O projeto destaca a criação de um componente de entrada de dados elegante, centralizado na tela e com um botão de ação integrado de forma fluida ao design.

## Tecnologias Utilizadas

* HTML5: Estruturação do campo de busca (`<input>`) e contêiner do componente.
* CSS3: Estilização de bordas arredondadas, sombras, transições e paleta de cores sutil.
* Flexbox: Alinhamento e centralização perfeita do componente na viewport, além da organização interna entre o campo de texto e o ícone de lupa.

## Funcionalidades e Técnicas Aplicadas

* Design de Pílula (Pill-shaped): Uso de `border-radius` avançado para criar um formato arredondado contínuo e orgânico.
* Botão Integrado: Botão de busca com ícone de lupa acoplado diretamente à extremidade direita da caixa de entrada de texto.
* Efeito de Profundidade: Aplicação de `box-shadow` suave sob a barra de busca para criar um efeito tridimensional sobre o fundo colorido.
* Feedback Visual: Configuração de estados de interação (como mudanças sutis ao focar no campo ou passar o mouse sobre o botão de pesquisa).
